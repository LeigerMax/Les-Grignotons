## Instructions

1. **Placez vos photos** dans les dossiers correspondants
2. **Nommez les fichiers** exactement comme indiqué ci-dessus
3. **Format recommandé** : JPG, format carré ou 4:3, minimum 800x800px
4. **Qualité** : Privilégiez des photos nettes et bien éclairées

## Exemple de noms de fichiers

- `creme-blanc-satin.jpg` pour "Crème et blanc (satin)"
- `agouti-argente.jpg` pour "Agouti argenté (argenté satin)"
- etc.

## Si une image manque

Si une image n'est pas trouvée, un placeholder sera affiché automatiquement.

## Catégories en construction

Les catégories "Poils courts" et "Alpaga" afficheront un message "Page en construction" jusqu'à ce que vous ajoutiez les couleurs dans le fichier `src/lib/data/colors.ts`.
